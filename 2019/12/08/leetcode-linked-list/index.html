复制带随机指针的链表 leetcode 138

方法一：使用 map<旧地址，新地址>

方法二：不用 map

  • 插入：每个旧节点后面插入一个自身的“复本”
  • 复制 random 指针
    • 一个旧节点 a 的复本是 a->next
    • a->ranom 的复本是 a->random->next
    • 新节点的random 指针 a->next->random = a->random->next (空值单独判断)
  • 拆分
    • 旧节点链表是奇数项
    • 新节点链表是偶数项
